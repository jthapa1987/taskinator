## 👤 Author: Jeevan Thapa  
A fast, intuitive task‑tracking app that organizes your workflow into clear, actionable stages.

---

## 🚀 Key Features
- **Dynamic Task Creation** — Add tasks with custom names and types (Print, Web, Mobile).  
- **Drag‑and‑Drop Workflow** — Move tasks between *To Do*, *In Progress*, and *Completed*.  
- **Full CRUD Support** — Edit or delete tasks instantly.  
- **Local Persistence** — Tasks auto‑save to `localStorage` and reload on startup.  
- **Responsive UI** — Clean layout, custom color palette, and mobile‑friendly design.
